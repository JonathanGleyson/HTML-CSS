Obs: Listas ordernadas é uma ordem dos itens que você deseja. Para criar uma lista ordenada igual a de cima, você começa com a tag <ol> para delimitar a lista e com a tag <li> para identifcar e dar ordem para cada item da lista. Caso deseja personalizar sua lista mudando a formatação da contagem, é só colocar a propriedade type="" dentro da tag <ol> e escolher o formato ou caso queira começar por um certo número da lista, é só colocar a propriedade start= "" dentro da tag <ol> também.

Obs: Listas não ordenadas é lista com marcadores, que são aquelas onde a ordem dos itens não influenciará. Para criar uma lista não ordenada, vamos usar a tag <ul> para demilitar e a tag <li> para criar cada um dos seus itens internos. Caso queira mudar o marcador padrão (bolinha preta), você pode mudar colocando a propriedade type="" dentro da tag <ul> e mudando com por disc, circle ou square.
